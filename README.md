# react-native-turbo-tracking-transparency

A React Native Library for interacting with the tracking API for ios iOS

## Installation


```sh
npm install react-native-turbo-tracking-transparency
```


## Usage


```js
import { multiply } from 'react-native-turbo-tracking-transparency';

// ...

const result = multiply(3, 7);
```
